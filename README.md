# Gestor de Tareas Pendientes (To-Do List)

## Versión
**1.0**

## Descripción
Aplicación de consola para gestionar tareas pendientes, organizadas por tipo (Trabajo, Personal, Estudio), prioridad (ALTA, MEDIA, BAJA) y fechas límite.

## Alcance

### ✅ Qué hace
- Crear tareas con título, descripción, fecha límite y prioridad
- Clasificar tareas por tipo (Trabajo, Personal, Estudio)
- Marcar tareas como completadas
- Filtrar tareas por categoría o estado
- Mostrar tareas pendientes ordenadas por prioridad
- Buscar tareas por palabra clave
- Notificaciones de tareas próximas a vencer y vencidas

### ❌ Qué NO hace
- No tiene interfaz gráfica (solo consola)
- No persiste datos en base de datos real
- No envía notificaciones reales (solo simulación)

## Tipos de Tarea

| Tipo | Atributos adicionales |
|------|----------------------|
| Trabajo | Proyecto, Cliente, Horas estimadas |
| Personal | Lugar, Requiere compañía |
| Estudio | Materia, Calificación esperada |

## Prioridades

| Prioridad | Descripción |
|-----------|-------------|
| ALTA | Urgente, debe hacerse pronto |
| MEDIA | Importante pero no urgente |
| BAJA | Puede esperar |
